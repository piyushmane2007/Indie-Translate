# Indie-Translate 🌐

Indie-Translate is a simple language translation web application built using Flask and Python's `translate` library.

The app allows users to translate text between several Indian languages, making it easy to convert text from one language to another through a clean and simple interface.

## Features

- Translate text between Indian languages
- Support for English, Hindi, Marathi, Bengali, Telugu, Tamil, Gujarati, Urdu, Kannada, and Malayalam
- Simple and responsive user interface
- Instant translation results
- Built using Flask and Python

## Tech Stack

- Python
- Flask
- HTML
- CSS
- translate library
